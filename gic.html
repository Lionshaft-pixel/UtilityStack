Grayscale Image Converter using Python

(Tkinter + OpenCV)

This Python program lets users select an image from their computer and automatically converts it into a grayscale (black & white) version. It uses Tkinter to create a simple graphical interface and OpenCV for image processing. When the user clicks the Open button, the program loads the chosen image, processes it, and displays the grayscale result in a new window.
